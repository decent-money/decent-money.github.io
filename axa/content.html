Allies X Axis (AXA) Alignment Chart

  • You can add 3 main types of content to each cell:
    • X accounts: Type an X handle including the @. Displays the avatar (if possible, or a placeholder).
    • Plain text: Type any text you want to display. Keep it short.
    • Images (+label): Paste an image from your clipboard (NOT image URL addresses, actual image data). After pasting image, add an optional custom label.
  • Labels for columns and rows also editable if you want to make a custom alignment chart (or click the FMK Labels button for a quick format change).

Note: Sometimes X avatars won't load from unavatar.io so a placeholder from dicebear.com will be displayed instead.
